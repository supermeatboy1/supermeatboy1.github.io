Who am I...

  I’m a second year computer science student who loves diving into different programming languages like Scala, Erlang, JavaScript, Rust, HTML, and CSS. Whether it’s building solid backend systems, handling concurrency, or tweaking UI elements, I enjoy tackling problems from different angles. I like experimenting with functional, concurrent, and object-oriented programming, which helps me adapt to all kinds of projects.


  Lately, I’ve also been getting into graphics programming, learning how rendering works under the hood. I focus on writing clean, efficient, and maintainable code, and I’m always up for picking up new tech and improving my skills. I thrive in fast-paced environments where I can build cool, impactful stuff.

Projects:

FPS Game
(Software Rendered)


A stripped-down Doom / Wolfenstein-style shooter with pseudo-3D software rendering. Built to learn the basics of rendering from scratch.

RFID Attendance Tracker


An automated attendance system for a school event using RFID scanners and a PostgreSQL database. Helped streamline check-ins and learned some valuable database skills.

Java 3D Software Renderer


A basic 3D renderer built to understand the rendering pipeline and apply linear algebra concepts. Loads a 3D OBJ model with a texture and displays it. Pure software rendering, no GPU required.

Computer Lab Manager


A system to monitor and control lab computers, track usage time, bill users, and allow admins to message, send files, and remotely control machines.

Codequest
(Programming Learning Platform)


A lightweight coding challenge platform using Cowboy for web serving and Erlang processes to run user-submitted code in Docker, comparing outputs against test cases. Players are rewarded based on test cases they've correctly answered.

BakeTrack
(Inventory System)


A .NET + SQL Server app to track ingredients, purchases, and stock levels, with alerts for low inventory to ensure smooth bakery operations.
